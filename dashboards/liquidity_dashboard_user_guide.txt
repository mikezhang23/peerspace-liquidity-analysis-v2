# Liquidity Dashboard User Guide

## How to Use:
1. Select a metro from dropdown to filter all views
2. Click on map to select specific metro
3. Hover over charts for detailed tooltips

## Dashboard Descriptions:
- Executive Summary: High-level overview
- Metro Deep Dive: Detailed metro analysis
- Predictive Analytics: ML predictions
- Action Center: Recommended actions

## Metrics Explained:
- Liquidity Score: 0-100 measure of market health
- Conversion Rate: % of searches that book
- Supply/Demand: Balance of venues to searches
